# 🌿About “Healing Space”🌿

### Healing Space는 “디지털 힐링 & 마인드 케어”를 비전으로 삼고 개발 중인 웹 서비스 입니다.

이를 비전으로 삼아 지금까지 다음과 같은 서비스를 개발하였습니다.

- AI 다이어리 서비스
- AI 고민 상담 서비스
- AI 심리 상담 챗봇 서비스
- 힐링 스토어 매칭 서비스
- 힐링 커뮤니티 운영(힐링 메시지 & 힐링 뮤직 공유 페이지)

앞으로는 다음과 같은 서비스를 개발할 예정입니다. 

- 웹뷰 모바일 어플리케이션 개발
- 힐링 메시지 카카오톡 전송 서비스
- 유저 간 친구 추가 및 네트워킹 시스템 구축
